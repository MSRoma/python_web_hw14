Запуск застосунку fastapi та баз данних postgres і redis в docker контейнері
виконується командою 'docker compose up -d'
в командный строці з кореневої папки проекту

Оновлення змін проекту в docker контейнері
виконується командою 'docker compose build'
в командный строці з кореневої папки проекту


Після запучку застосунку перейти до головної сторінки можна за посиланням
http://localhost:8000/docs#/
